Executive Summary

웹은 클라이언트-서버 모델을 기반으로 작동합니다. 사용자의 브라우저(클라이언트)가 요청을 보내면 서버가 HTML, CSS, JS 데이터를 응답하며, 이 과정은 DNS와 HTTP 프로토콜이라는 규칙 위에서 이루어집니다.

클라이언트 (Client)

💻
  • • 웹 브라우저 (Chrome, Safari 등)
  • • 정보를 요청하는 '사용자' 기기
  • • URL을 통해 서비스에 접근

서버 (Server)

🗄️
  • • 데이터를 저장하고 처리하는 컴퓨터
  • • 클라이언트 요청에 응답(Response)
  • • 웹 리소스를 호스팅하고 관리
